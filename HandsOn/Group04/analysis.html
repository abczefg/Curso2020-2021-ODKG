Analysis of data attributes

Creation Date

Type : Date
Range : 2012-07-02 to 2015-06-30
Description : Creation date

Purchase Date

Type : Date
Range : Ok
Description : Purchase Date. 17422 null values readen and nonsense dates.

Fiscal Year

Type : Couple of years
Range : From 2012-2013 to 2014-2015
Description : Fiscal Year

LPA Number

Type : Plain text
Range : /
Description : Leveraged Procurement Agreement Contract Number. Numbers, tirets and also includes characters. 253607 null values.

Purchase order number

Type : Plain text
Range : /
Description : includes only numbers, sometime characters and also '-'. Sometimes the same as requisiton number. No null values.

Requisition number

Type : Plain text
Range : /
Description : Fiscal designated unique number of the associated Requisition. 330138 null values.

Acquisition Type

Type : Plain text
Range : 5 possible values : IT Goods, IT Services, IT Telecommunications, NON-IT Goods, NON-IT Services. No null values.
Description : Information about what the acquisition is related to.

Sub-Acquisition type

Type : Plain text
Range : 25 possible non-null values and 276170 null values.
Description : For Non-IT Services Acqusition Type, user must also indicate what type of Non-IT services they are purchasing.
What the contract is related to (Personal services, public works, etc.)
