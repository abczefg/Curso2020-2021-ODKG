Resource Naming Strategy

The URL of the website from which you can fetch the Data is http://eit-upm-opendata.com

Notice

  • get list of ids of Notices : /notices
  • get notice with identifier of notice-id : /notice/notice-id
  • get contract id of notice-id : /notice/notice-id/#contract
  • get CAN id of notice-id : /notice/notice-id/#can
  • get type of notice : /notice/notice-id/#type
  • get if notice is cancelled or not : /notice/notice-id/#cancelled
  • get if notice has mutiple CANs : /notice/notice-id/#multiple-cans
