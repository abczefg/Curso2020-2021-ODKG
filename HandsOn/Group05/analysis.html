Data properties of ontology

After cleaning, the dataset contains 21 columns and 7587 rows.

There are two classes; CAE and Notice.

The data contains the following properties;

  • CPV - which is the main Common Procurement Vocabularycode of the main object of the contract and is of type double and belongs to column CPV.
  • Contract Type - which is the type of contract with the values: W "Works", U "Supplies", S "Services" of type string and belongs to column TYPE_OF_CONTRACT
  • Country Code - which is the code of the "Country" for the first listed authority and is of type string and belongs to column ISO_COUNTRY_CODE
  • Date - which is the date of the contract notice and is of type DateTime and belongs to column DT_DISPATCH
  • Duration - which is the duration of the contract and is of type double and belongs to column DURATION
  • ID Type - which is the Standard form number and can be used to select which directive the notices fall under and is of type integer and belongs to column ID_TYPE
  • Main Activity - which is the category of the contract (e.g. economics, health, etc.) and is if type string and belongs to column MAIN_ACTIVITY
  • Multiple CAE - which shows if there is more then one contracting authority or entity and is of type boolean and belongs to column B_MULTIPLE_CAE
  • CAE Name - which is the "offical name" of the CAE and is of type string and belongs to column CAE_NAME
  • National ID - which is "National registration number" e.g. VAT number for utilities and is of type string and belongs to column CAE_NATIONALID
  • Postal code - which is the "postal code" of the CAE and is of type string and belongs to column CAE_POSTAL_CODE
  • TOP - which is Type of procedure: AWP “award without prior publication of a contract notice”, COD “competitive dialogue”, NEC/NEG “negotiated with a call for competition”, OPE “open", RES “restricted” and is of type string and belongs to column TOP_TYPE
  • Town - which is the "town" of CAE and is of type string and belongs to column CAE_TOWN
  • CAE Type - which is the type of contracting authority: 1 “Ministry or any other national or federal authority, including their regional of local subdivisions”, 3 “Regional or local authority”, 4 “Water, energy, transport and telecommunications sectors”, 5 "European Union institution/agency", 5A "other international organisation", 6 "Body governed by public law", 8 "Other" N "National or federal Agency / Office", R "Regional or local Agency / Office", Z “Not specified" and is of type string and belongs to column CAE_TYPE
  • Value - which is the CFC value, in EUR, without VAT, if the value variable is missing, the framework value is used instead (overwritten by humanmade estimates of values for large value contracts which seemed to be incorrect and is of type double and belongs to column VALUE_EURO_FIN_2
  • Cancelled - which shows if the contract is cancelled or not and is of type boolean and belongs to column CANCELLED
  • EU Funded - which shows wether or not the contract is funded by the EU and is of type boolean and belongs to column B_EU_FUNDS
  • Electronic - which shows if an electronic auction has been used or not and is of type boolean and belongs to column B_ELECTRONIC_AUCTION
  • GPA - which shows if the contract is covered by the Government Procurement Agreement or not and is of type boolean and belongs to column B_GPA.
