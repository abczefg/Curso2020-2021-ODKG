Application Requirements

Minor Contracts Madrid Town Hall 2020

The application will allow the user to find the minor contracts given to companies or organizations in 2020. The user will be allowed to filter by the recognition number of the contract, contractor's N.I.F, date of the contract, price range, section or department, contractor's name and type of the contract. The output will be all the information around the contract and the contractor in a table format. Besides filtering, the user will be allowed to sort ascending and descending. Some graphic plots will be also generated like the evolution in time of the public spending for each type of contract. The aim of this application is to provide citizens with public contracts in a more transparent manner, so the user will be able to know how the politicians spend their money.
